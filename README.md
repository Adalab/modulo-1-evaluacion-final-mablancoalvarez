

# Anonymous Proxy

Bienvenidos a la creación de una landing Page, con una estructura muy diferenciada:

-Header
-Hero
-Main
-Section
-Footer


## Construido con 🛠️

* HTML5
* SASS
* GULP
* GIT

Dentro del proyecto se han usado animaciones, transiciones, grid y flexbox para toda la estructura.
